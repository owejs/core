expose

Exposes objects as another value when serialized. expose uses the expose property of object resources internally.

Parameters:
Name Type Attributes Description
obj object

The object to expose.
val any <optional>

The value that should be used instead of obj when serialized. If not specified and obj is an Error, obj.message will be made enumerable and obj will get itself as its exposed value.
